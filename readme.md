# 2 суммы из leetcode
дан массив чисел [0, 1, 2...] необходимо найти 2 числа, которые будут равны target (любое целое число)

## 1 в лоб
просматриваем все комбинации смотрим какие комбинации дают таргет
O(N^2)

## 2 "самый" быстрый случай с картой
Закидываем наши числа в карту для O(1) просмотра
n1 + n2 = target, 
target - n2 = n1
Для каждого элемента мапы считаем комплимент.
без перебора O(вообще) становиться O(n)

## 3 2 указателя.
Для отсортированного массива сложность O(NLogN)
1 указатель остается на месте, другой двигается.g
